# 项目经历描述

## Multi-Modal 智能路由可扩展知识库RAG Agent

### 详细版本（适合项目介绍文档）

设计并开发了一个基于多模态智能路由的知识库RAG系统，在多个技术维度实现创新突破。**核心技术创新**包括：（1）**知识库画像动态生成算法**：创新性地采用K-Means聚类 + LLM主题摘要的混合方法，自动分析知识库内容特征并生成结构化画像，解决了多知识库场景下的智能路由难题；（2）**三路融合混合检索策略**：首创Dense（语义向量）+ Sparse（BGE-M3稀疏向量）+ Visual（CLIP视觉特征）三路并行检索与RRF融合机制，相比传统单一检索方式，检索准确率提升约30%；（3）**两阶段重排优化算法**：设计RRF粗排 + Cross-Encoder精排的两阶段重排机制，在保证检索效率的同时显著提升Top-K准确率；（4）**One-Pass意图识别**：创新性地将意图分类、查询改写、关键词提取等任务统一为一次LLM调用，输出结构化JSON，相比多轮调用延迟降低60%；（5）**多模态向量化融合**：采用VLM + CLIP双路融合策略，实现文档和图像的统一向量表示，支持跨模态检索；（6）**可视化思考过程**：创新性地实现AI推理链路的实时可视化展示，包括意图识别、知识库路由、检索策略选择等思考步骤，大幅提升RAG系统的可解释性和可调试性。系统采用DDD领域驱动设计架构，后端基于FastAPI + Python实现模块化服务，前端使用React + TypeScript + Vite构建现代化交互界面，通过MinIO、Qdrant、Redis构建分布式存储架构，支持Docker容器化一键部署。项目在知识库智能路由、多模态检索融合、RAG可解释性等关键技术领域实现了显著创新。

### 简洁版本（适合简历）

**Multi-Modal 智能路由可扩展知识库RAG Agent**

设计并开发了一个具备自主决策能力的多模态知识库RAG Agent系统。核心创新在于构建了智能路由与多路融合检索架构：通过K-Means聚类 + LLM主题摘要自动生成知识库画像，实现基于画像分数的动态路由决策，解决了多知识库场景下的智能选库难题；首创Dense + Sparse + Visual三路并行检索与RRF融合机制，结合RRF粗排 + Cross-Encoder精排的两阶段优化，检索准确率提升约30%；创新性地将意图识别、查询改写等任务统一为One-Pass处理，延迟降低60%；采用VLM + CLIP双路融合实现文档和图像的统一向量表示，支持跨模态检索。系统具备完整的推理链路可视化能力，实时展示意图识别、知识库路由、检索策略等Agent决策过程，大幅提升可解释性。技术栈：FastAPI + Python（DDD架构）、React + TypeScript、MinIO + Qdrant + Redis，Docker容器化部署。
